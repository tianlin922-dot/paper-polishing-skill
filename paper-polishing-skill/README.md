# paper-polishing-v1 Skill

This package converts the original `paper-polishing-v1` README into a ChatGPT Skill-style folder.

## Files

- `SKILL.md`: the actual skill instruction file.
- `resources/paper_polishing_prompt_library.md`: the original full prompt framework, kept as a reference library.
- `examples/`: reserved for future before/after examples.

## Suggested first test

Ask ChatGPT:

> 请使用 paper-polishing-v1，对下面这段中文学术段落做英文期刊风格深度重写。

Then paste one paragraph from a manuscript.
